# 🎬 Movie List

Movie List is a web application that allows you to build and manage your personal movie directory. You can keep track of the movies you have watched, mark your favorites, and discover films using IMDb data. The goal of this project is to provide a **simple personal catalog of movies you have seen**.

## 🚀 Deployment

This project can be deployed very easily on platforms such as **Vercel** or **Netlify**, since the application is built with **Vite**, it only requires a simple static build.

---

# ✨ Features

- 📚 Create your personal movie list
- ⭐ Mark movies as **favorites**
- 👁️ Mark movies as **watched / not watched**
- 👁️ Filtering on movies
- 🔎 Search and add movies using **IMDb data**
- ☁️ User authentication
- 💾 Cloud database for storing your movie list

---

# 🛠 Technologies

This project is built with:

- **Vite**
